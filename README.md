# poly-bridge-frontend

Poly Bridge web application based on Vue

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes file

```
yarn run lint
```
